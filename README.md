## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and visit the URL shown in the terminal (usually `http://localhost:5173`)

## Customization

You can customize the love messages by editing the `loveMessages` array in `src/App.jsx`. Feel free to add your own personal messages!

## Build for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` folder, ready to deploy or share!
